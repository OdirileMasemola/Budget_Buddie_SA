# 📱 Budget Buddie SA

## 📌 Overview
Budget Buddie SA is a personal budgeting Android application designed to help users track their spending, manage their finances, and stay within their budget.

The app focuses on simplicity, usability, and helping users build better financial habits.

---

## 🎯 Purpose
The purpose of this app is to provide students and young adults with a simple and effective way to:
- Track daily expenses  
- Organise spending into categories  
- Monitor budget goals  
- Improve financial awareness  

---

## 🚀 Features

### ✅ Implemented (Prototype)
- User login system  
- Add and manage expenses  
- Create and manage categories  
- View expense history  
- Set budget goals  

### 🔜 Planned (Final Version)
- Graphs for spending trends  
- Budget progress dashboard  
- Gamification (badges & rewards)  
- Notifications and reminders  

---
